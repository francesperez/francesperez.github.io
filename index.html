Recent Projects

Bloagbea

Blogbea is a simple, minimalist blogging website where users can create an account and post short snippets of whatever is on their mind. Users can edit and delete their individual posts, while visitors are able to view all posts without an account. On the backend, this project utilizes MySQL and Java with Maven using the Spring Boot Framework and Thymeleaf.

CuteWeather

CuteWeather uses OpenWeather’s free API to forecast weather data by searching for a specific location. CuteWeather’s frontend was made using a combination of HTML, CSS, Javascript, jQuery, and Bootstrap 5. A unique feature of this application is that not only can you type an address into a search bar, but drag a pin across a map to update the forecast for the week.

Venued

Venued is social media platform where users can post tips, ask questions, and answer questions based on venues that have been searched for. Users can create accounts, edit or delete comments, and up/down vote tips or answers. Venued was made using the Yelp and PreditHQ APIs and utilizes JDBC and a Google SMTP server for various functionalities. The platform was built using Springboot, Thymeleaf, JavaScript, jQuery, HTML, and CSS for a seamless user experience.
